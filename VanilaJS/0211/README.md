## 2.5
#### If, else
- 조건문
- If문의 조건 (condition) 에서 **==**는 강제 형변환을 시도, **===** 은 형변환을 하지 않는다
    - 따라서 3 == '3' 은 둘을 강제로 같은 타입으로 형변환시키기 때문에 true가 나오지만, 
    - 3 === '3' 은 int와 string의 비교이기 때문에 false가 나옴
- else if은 그냥 else if이라 쓴다 (python처럼 elif이 아님)
- 조건문에서 같지않다 는 **!==**

#### 논리연산자 and, or
- &&, ||
- and는 교집합, or은 합집합
